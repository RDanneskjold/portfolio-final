Projects

YouTube Database [YTDB]

YouTube's search algorithm is largely based on a channel's traffic, which results in a stagnant list of suggestions. Furthermore, a consumer must rely on the channel creator's description of their content which is oftentimes inaccurate or nonsensical. YouTube Database provides search results with a preference for channels that have been rated or reviewed by our users. This allows anyone visiting the site to learn more about channels they might watch and dive deeper into a list of channels related to their interests.

I was project manager for our five-person team and developed a significant amount of YTDB's UI using React, context, and handling API requests and responses to our backend, using Node.

Bonjour!

Bonjour! is a mock language learning tool which uses the spaced repetition algorithm for quicker, tailored learning. The app is built using React, Node and Postgres and is deployed via Heroku for the api and Zeit for the client.

This app was a two-person project, created during the Thinkful course. I created the backend using Node and Postgres, implementing the spaced repetition algorithm with a singly linked list.

Philadelphia Courts Language Access Plan
First Judicial District of PA Language Access Plan

I cowrote the Language Access Plan (LAP) for the First Judicial District (FJD) of Pennsylvania with Deputy Court Administrator, Janet Fasy, and the Honorable Judge Ida Chen. Required by the federal government for agencies to receive federal funds, this project was the first LAP of any judicial district in the state of Pennsylvania. It set the standard by which other counties are measured. This took considerable research, writing, and project management skills to accomplish.

Korean Adoptee Community
Korean Adoptees Facebook Group

For the past five years, I have been an admin for the "Korean Adoptees" Facebook group which contains 4,200+ members. Membership includes Korean adoptees from around the world, of all age ranges and experiences. This is not a group to post selfies or self-promote.

I am also actively involved in the Korean Adoptee Association of Philadelphia and was the Philadelphia Team Leader for Connect-A-Kid, a Korean Adoptee mentorship program.
